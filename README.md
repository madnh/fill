# Fill data to template

### Create project

Run `Create Project.bat` file then input project name.

**Project's structure**

```
+==[Sample Project ]
    |--->    0. Profiles.json
    |--->    1. Template.html
    |--->    2. Wrapper.html
    \--->    3. Output.html
```

### Filling

Run `Fill.bat` file then input project name.
